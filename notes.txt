HTML - Hyper text markup language       <head> content </head>

self closing tag - hr, meta, br, img, input

==> It is not a programming language
==> it is a markup language
==> Every website can start with HTML

1) Heading - h1 to h6

2) Paragraph - p

3) hr

4) break tag - br

5) Text formatting - u, i, b or strong, sub, sup and mark

6) Anchor tag(linking) - a

7) Image tag - img

8) List - ordered list and unordered list

9) Tables - tr, th and td

10) Forms - Input, placeholder, type(important), required, button and label

11) Media Elements - video and audio

12) Sematic tags - Header, footer, section, nav and main

13) division tag - div


CSS - Cascading Style Sheet

1) Types of CSS
==> inline CSS
==> Internal CSS
==> External CSS
inline > Internal > External

2) CSS Syntax
selector{
    property: value;
}

3) Types of Selector
==> Element Selector - (h1) (h2) (p)
==> Class Selector - (.) - (Multiple) - no of times we are using the same class name
==> Id Selector - (#) - (Unique) - only one time we are using the same id name
==> Universal Selector - (*) - to be apply the entire site
==> Group Selector - class or id Selector contents different tags, we wanna change the same style means we go with group selector
